----------------- DinoGame --------------------
---
A game inspired by the iconic chrome dino game  

How to play
---
Avoid touching the obstacles for as long as you can.  
The controls are very simple, just as the original:  
ArrowUp --> Jump  
ArrowDown --> Duck
